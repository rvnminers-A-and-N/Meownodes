What is Ravennodes?

Ravennodes estimates the size of the Ravencoin network by finding all the reachable nodes These are the nodes that accept incoming connections. This method will not list all full nodes because not all nodes have an open port that can be probed using Ravennodes. These nodes are either behind firewalls or configured to not listen for connections.
